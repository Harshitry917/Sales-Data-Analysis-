# Sales Data Analysis Project

## 📊 Overview
This project analyzes sales data using Excel and Power BI to generate business insights.

## 🛠 Tools Used
- Excel
- Power BI

## 📈 Analysis Performed
- Region-wise revenue and profit analysis
- Category-wise performance
- Product-wise analysis
- Month-wise trend analysis

## 🔍 Key Insights
- East and West regions performed best
- Electronics category generated highest profit
- Laptop is the most profitable product
- Seasonal trends observed in monthly sales


## 📁 Files Included
- Excel dataset
- Power BI dashboard file
